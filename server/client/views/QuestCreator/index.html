Quest Creator Download

The Quest Creator is a package that contains all the tools needed to create and test quests for Raining Chain.

Here's a brief overview of the software:

  • Setting: Modify general quest information such as the name, description and author.
  • Variable: Create quest variables. Every player has their own copy of variables. Modify them in events to save progression.
  • Event: The logic of the quest. The software provides documentation for each functionality. Ex: Teleporting player, adding items, spawning monsters.
  • Map: Create maps using the software Tiled and specify what monsters to spawn by default.
  • Item: Create new items, their options and their icon.
  • Dialogue: The software provides a powerful dialogue system. Each dialogue can offer options that trigger other dialogues or custom events.

Installing

  • Download.
  • Unzip.
  • Move the rc folder at C:\rc
  • Install nodejs_server.msi at C:\Program Files (x86)\nodejs
  • Install map_editor.exe at C:\Program Files (x86)\Tiled
  • Open the Quest Creator via the shortcut at C:\rc\QuestCreator.exe.
  • Check tutorials.
